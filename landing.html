你的私人家庭影院

一个命令启动，手机、平板、电脑打开浏览器即可观看。
自动转码，支持所有主流视频格式。

为什么选 LocalCinema

⚡

零配置启动

单个二进制文件，ffmpeg 不存在时自动下载。无需安装任何依赖，开箱即用。

🎬

全格式支持

MP4 直接播放，MKV / AVI / MOV / WebM / WMV / FLV 自动 HLS 转码串流。

📱

多设备访问

局域网内任何设备打开浏览器即可观看，无需安装 App。手机和桌面端自适应布局。

💫

硬件加速转码

macOS 上利用 VideoToolbox 硬件编码，转码快速且 CPU 占用低。

💾

智能缓存

转码结果、封面、时长持久缓存，相同视频再次播放秒开。支持一键清空缓存。

▶️

播放进度记忆

自动保存播放位置，下次打开同一视频时提示从上次位置继续观看。

🌓

深色 / 浅色主题

自动跟随系统深色模式，也可手动切换。护眼观影，随心选择。

🔍

搜索与视图

首页实时搜索过滤，列表与平铺视图一键切换。自动生成封面和时长标签。

🔒

隐私优先

纯本地运行，无需联网（ffmpeg 下载后离线可用）。你的影库只属于你。

三步开始

1

安装

下载对应平台的二进制文件，或使用 Go 安装：
go install github.com/raojinlin/localcinema@latest

2

启动

指定你的视频目录运行：
localcinema -dir ~/Movies
首次启动自动下载 ffmpeg，默认监听 8080 端口，可通过 -port 修改。

3

观看

在同一 Wi-Fi 下，用手机或平板打开终端提示的地址即可浏览和播放视频。

支持的格式

常见视频格式均可播放，非 MP4 格式自动通过 HLS 转码串流

.mp4 .mkv .avi .mov .webm .m4v .wmv .flv
